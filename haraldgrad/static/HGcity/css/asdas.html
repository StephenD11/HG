Урок 9: Функции в Python

1. Что такое функция?

Функция — это блок кода, который можно вызвать несколько раз.

# Определение функции
def greet():
    print("Привет! Добро пожаловать в программу.")

# Вызов функции
greet()

2. Функция с аргументами

# Функция с аргументами
def introduce(name, age):
    print(f"Меня зовут {name}, мне {age} лет.")

introduce("Анна", 25)
introduce("Иван", 30)

3. Функция с возвращаемым значением

# Функция, возвращающая значение
def add_numbers(a, b):
    return a + b

result = add_numbers(5, 3)
print("Результат сложения:", result)

4. Аргументы по умолчанию

# Аргумент по умолчанию
def greet(name="Гость"):
    print(f"Привет, {name}!")

greet()
greet("Анна")

Что нужно запомнить:

  • Функция — это блок кода, который можно вызывать несколько раз.
  • Аргументы позволяют передавать данные в функцию.
  • return позволяет вернуть значение из функции.
  • Аргументы по умолчанию используются, если значение не передано.
